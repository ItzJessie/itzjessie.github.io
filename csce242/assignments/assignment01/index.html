Marvel Rivals contains two popular game modes. The first is quickplay, which is a 6v6 objective based game mode. The second is competetitive, this is a 6v6 objective based game mode but, it also provides a ranking system for players to strive for the top 1%. These game modes can be highly intense with each player potentially changing the game outcome!

Marvel Rivals is supported globally. It can support the following systems including PCs, laptops, Ps5, XBOX, tablets, and smartphones. Marvel Rivals is optimized with low requirements which gives more potential players to enjoy the game. Marvel Rivals provides five global server regions which enables, the upmost support for gaming experience to their players.

  • Convoy
  • Domination
  • Convergence
  • Conquest
